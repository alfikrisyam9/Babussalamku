# Babussalamku

## Deskripsi Project
Babussalamku adalah website sederhana Pondok Pesantren Babussalam yang dibuat untuk memenuhi tugas mata kuliah Web Programming.  
Website ini bertujuan sebagai media informasi dan pendaftaran santri baru secara online, sehingga memudahkan masyarakat dalam memperoleh informasi dan melakukan pendaftaran.

Website memiliki fitur:
- Halaman beranda
- Halaman profil pesantren
- Halaman pendaftaran santri (terhubung database)
- Sistem login admin untuk melihat data pendaftar

## Teknologi yang Digunakan
- **HTML** : Struktur halaman website  
- **CSS** : Desain dan tampilan website  
- **Bootstrap** : Framework frontend untuk membantu tampilan responsif  
- **PHP** : Backend untuk memproses form dan login admin  
- **MySQL** : Database untuk menyimpan data pendaftaran santri dan akun admin

## Cara Menjalankan Project
1. Install dan jalankan **XAMPP**
2. Simpan folder project di:
C:\xampp\htdocs\Babussalamku
3. Jalankan Apache dan MySQL
4. Akses website melalui browser:
http://localhost/Babussalamku/
